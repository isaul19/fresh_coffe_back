# FRESH COFFE (FRONT)

## Requerimientos

-   docker
-   php (v8.0+)
-   composer

## Pasos

Clonar repositorio

```bash
  git clone https://github.com/isaul19/fresh_coffe_front.git
```

Dirigirse al directorio

```bash
  cd ./fresh_coffe_front
```

Instalar dependencias

```bash
  ./vendor/bin/sail composer install
```

Levantar servidor local

```bash
  ./vendor/bin/sail up
```
