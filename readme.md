# FRESH COFFE (BACKEND)

## Requerimientos

-   docker
-   php (v8.0+)
-   composer

## Pasos

Clonar repositorio

```bash
  git clone https://github.com/isaul19/fresh_coffe_backend.git
```

Dirigirse al directorio

```bash
  cd ./fresh_coffe_backend
```

Instalar dependencias

```bash
  ./vendor/bin/sail composer install
```

Levantar servidor local

```bash
  ./vendor/bin/sail up
```
